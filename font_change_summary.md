# تغيير الخط إلى Cairo - ملخص التحديث

## التغيير المطلوب: ✅ تم بنجاح

تم تغيير خط الموقع من **GE SS Two** إلى **Cairo** في جميع أنحاء الموقع.

## التعديلات التي تمت

### 1. تحديث رابط الخط
**قبل:**
```html
<link href="https://fonts.googleapis.com/css2?family=Montserrat:wght@300;400;500;600;700;800;900&display=swap" rel="stylesheet">
<link href="https://fonts.cdnfonts.com/css/ge-ss-two" rel="stylesheet">
```

**بعد:**
```html
<link href="https://fonts.googleapis.com/css2?family=Cairo:wght@300;400;500;600;700;800;900&display=swap" rel="stylesheet">
```

### 2. تحديث CSS العام
**قبل:**
```css
* {
  font-family: "GE SS Two","Montserrat",sans-serif!important;
}
```

**بعد:**
```css
* {
  font-family: "Cairo",sans-serif!important;
}
```
